Sobre mí

Ingeniería con criterio, no solo código

Soy un Desarrollador de Software con más de cinco años diseñando y construyendo productos digitales para negocios que necesitan resultados reales: velocidad, estabilidad y una experiencia de usuario que convierte.

Trabajo de extremo a extremo — desde la arquitectura de la base de datos hasta el último detalle de interfaz — combinando Vue.js, PHP y bases de datos como PostgreSQL y MySQL para entregar soluciones robustas y mantenibles.

Mi enfoque es simple: entender el negocio antes de escribir una línea de código, y construir software que siga funcionando bien mucho después de la entrega.

  • Código limpio y documentado
  • Entregas puntuales
  • Comunicación constante
  • Soporte post-lanzamiento
